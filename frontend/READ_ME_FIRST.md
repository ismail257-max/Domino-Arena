## 🚀 3-STEP SETUP:

### 1️⃣ Extract & Navigate
```bash

# Navigate into it
cd domino-arena-frontend-phase1
```

### 2️⃣ Install Dependencies
```bash
npm install
```
⏱️ Takes 2-5 minutes

### 3️⃣ Start Development Server
```bash
npm start
```
✅ Opens automatically at http://localhost:3000
